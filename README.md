# Bairamburguer - Sistema de Delivery Full-Stack
Sistema de gestão e delivery projetado para operação comercial real. Desenvolvido com foco em alta disponibilidade, escalabilidade e gestão ágil de pedidos.

## 🚀 Tecnologias
**Backend:** Java 17 + Spring Boot

**Frontend:** Next.js 14+ + Tailwind CSS

**Banco de Dados:** PostgreSQL 18

## 🏗️ Estrutura do Projeto
**/backend:** API RESTful e lógica de negócio.

**/frontend:** Interface do cliente e painel administrativo.

**/database:** Scripts de modelagem e estrutura SQL.
